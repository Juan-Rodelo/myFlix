Create a REST API with Node.js

Objective:

Server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Project: mongoDB, Express, Node.js, moongose

Requests (Business Logic) URL Endpoint HTTP Method Request Body Response Body
Get a list of all movies /movies GET None JSON
Get information about one specific movie by title; use %20 for spaces /movies/:Title GET None JSON
Return data about a director by name; use %20 for spaces /movies/Directors/:Name GET None JSON
New users registration /users POST JSON JSON
Put updates to user information /users/:Username PUT JSON JSON
Allow users to add a movie to their list of favorites /users/:Username/Movies/:MovieID POST None JSON
Delete movie from list of user's favorites /users/:Username/Movies/:MovieID DELETE None JSON
Deletes user from registration database /users/:Username DELETE None Message
